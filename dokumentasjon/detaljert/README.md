# Detaljert dokumentasjon – sokos-utleggstrekk

Denne mappen inneholder detaljert teknisk dokumentasjon av sokos-utleggstrekk, inkludert kode, flyt og arkitektur.

## Innhold

| Fil | Beskrivelse |
|-----|-------------|
| [arkitektur.md](arkitektur.md) | Detaljert arkitektur, lagdeling og kodestruktur |
| [flyt.md](flyt.md) | Detaljert prosesseringsflyt med sekvensdiagrammer |
| [trekk-transformasjon.md](trekk-transformasjon.md) | Forretningslogikk for transformasjon av trekkpålegg |
| [kode.md](kode.md) | Gjennomgang av de viktigste klassene og tjenestene |

## Forutsetninger

For å forstå den detaljerte dokumentasjonen bør du ha kjennskap til:
- Kotlin og JVM-baserte applikasjoner
- IBM MQ (meldingskø)
- PostgreSQL
- Ktor (Kotlin web-rammeverk)
- NAVs NAIS-plattform og GCP
